# AquabitOmer.forAviUtlTextEditor
AviUtlの簡単な制御文字をつけることができるテキストエディタです。



プロジェクトのタイトル
	AquabitOmer.forAviUtlTextEditor

プロジェクトの概要説明
	簡単なテキストの編集。
	平文のテキストのインポート
	AviUtlの制御文字をつけたテキストのエクスポート


必要条件
	windows10以上（だと思うよ！）

使用言語、環境、テクノロジー
	C#

使い方
	暫定版なので動画などを見てどのような機能なのか確認をお願い致します。

  ZIPから展開して好きな場所に保存してください。
